*Инструкция:*

Скачайте репозиторий
``` bash
git clone https://github.com/RomaTur/telegramMsg.git
```

Перейдите в папку и скачайте зависимости
``` bash
cd telegramMsg
npm i
```

### Запишите токен своего бота и id чата в config.json

Запустите сервер
``` bash
npm start
```
